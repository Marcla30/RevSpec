🚀 Démarrage Rapide

Option 1 : Double-cliquez sur index.html
Option 2 : Lancez un serveur local :
python3 -m http.server 8000
Puis ouvrez http://localhost:8000

🎨 Personnalisation

  • Images : Remplacez les SVG dans assets/images/voitures/
  • Couleurs : Modifiez les variables CSS dans style.css
  • Voitures : Ajoutez des objets dans le tableau cars de app.js
